# Tejaswi Portfolio

Personal portfolio website showcasing my projects, skills, and experience.

## Live Demo

- (https://tejaswi-portfolio-react.vercel.app/)

## Features

- Modern UI with animations and smooth section navigation
- Projects section with project details modal
- Skills marquee + highlighted skill cards
- Contact form powered by Formspree
- Resume download

## Tech Stack

- React (Vite)
- styled-components
- JavaScript, HTML, CSS

## Run Locally

```bash
npm install
npm run dev
```
